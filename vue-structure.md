```shell
$ npm init vue@latest			// projekt initialisieren -> eingabe des projektnamen
   cd projektname
$ npm install				// installation im projektordner
$ npm run dev				// starten im entwicklungsmodus

$ npm run build				// seite bauen
$ npm run preview			// vorschau anzeigen


Ordnerstruktur

project (src)
|
|- App.vue		// Hauptseite wie index.html + import lokaler komponenten
|- main.js		// Erstellung app + import globaler komponenten
|- /components
|  |- FoodItem.vue	// componente 
|
```
